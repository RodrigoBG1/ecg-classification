# ECG Classification — CLAUDE.md

## Project overview
Multi-label ECG classification on PTB-XL+ (21,799 12-lead recordings, 500 Hz).
Phase 1 complete: project scaffold + ResNet-1D (SE + stochastic depth) + data pipeline.
Planned phases: PatchTST, CNN-Transformer hybrid, ablation study.

## Architecture decisions
- Labels: 5-class binary vector [NORM, MI, STTC, CD, HYP] via SCP code → diagnostic_class mapping.
- Loss: BCEWithLogitsLoss with per-class pos_weight (handle class imbalance).
- Split: strat_fold column — folds 1-8 train, 9 val, 10 test. Do NOT shuffle or re-split.
- Normalization: per-lead z-score fitted on training set only via streaming algorithm (no OOM); stats saved to config/norm_stats.npy.
- Sampling rate: 500 Hz (5000 samples per recording). Records live in records500/.
- Primary eval metric: macro AUC-ROC (threshold-independent). F1-macro also tracked.

## ResNet-1D architecture (v2)
- Stem: Conv(12→64, k=15, s=2) → BN → ReLU → MaxPool(k=4, s=4) → 625 timesteps
- 4 stages × 2 blocks = 8 residual blocks total; channels [64, 128, 256, 512]
- Each block: Conv(k=7) → BN → ReLU → Conv(k=7) → BN → SEBlock1D → DropPath → + skip → ReLU
- Stochastic depth: drop_path_rate linearly scheduled 0 → 0.2 across stages
- Head: GlobalAvgPool → Dropout(0.3) → Linear(512, 5)
- ~8.9M parameters

## Key files
- `data/dataset.py`       — PTBXLDataset; reads wfdb records500/, applies transforms.
- `data/labels.py`        — SCP code parsing → binary label vector.
- `data/preprocessing.py` — Normalizer + augmentation (GaussianNoise, RandomAmplitudeScale, RandomCrop, RandomLeadDropout).
- `models/resnet1d.py`    — ResNet-1D with SEBlock1D and DropPath (logits out, no sigmoid).
- `training/trainer.py`   — Trainer with BCEWithLogitsLoss, AdamW, CosineAnnealingLR, Mixup, mixed precision.
- `training/metrics.py`   — compute_metrics() → dict with AUC-ROC, macro F1, and per-class breakdown.
- `scripts/preprocess.py` — Streaming normalizer, saves norm_stats.npy and pos_weight.npy.
- `scripts/train_resnet.py` — Main training entry point.
- `config/config.yaml`    — All hyperparameters (sampling_rate, batch_size, lr, drop_path_rate, etc.)
- `config/norm_stats.npy` — Per-lead mean/std (generated by preprocess.py, do NOT commit).
- `data/processed/pos_weight.npy` — Per-class BCE weights (generated by preprocess.py).

## Data location
Dataset root: `ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3/`
Required contents: `ptbxl_database.csv`, `scp_statements.csv`, `records500/`

## Running order
```powershell
# Step 1 — compute normalizer stats and pos_weight (streaming, ~7 min)
python scripts/preprocess.py `
    --data_path "ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3" `
    --output_path data/processed/ `
    --sampling_rate 500

# Step 2 — train ResNet-1D (100 epochs, early stopping patience=15)
python scripts/train_resnet.py `
    --data_path "ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3" `
    --norm_stats config/norm_stats.npy `
    --pos_weight data/processed/pos_weight.npy `
    --sampling_rate 500
```

## Conventions
- All functions have type hints (Python 3.10+).
- pathlib.Path everywhere — no hardcoded strings.
- Every module has an `if __name__ == "__main__"` smoke test.
- Model forward() returns logits; sigmoid/threshold applied only at eval time.
- Augmentations applied only to training split; PTBXLDataset handles this internally.
- PowerShell line continuation uses backtick (`), not backslash.
