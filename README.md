# ECG Classification — PTB-XL+

Multi-label ECG classification on the [PTB-XL](https://physionet.org/content/ptb-xl/1.0.3/) dataset,
comparing three deep learning architectures for 12-lead ECG signal classification.

## Architectures

| Model | Status |
|-------|--------|
| ResNet-1D | Implemented |
| PatchTST | Phase 2 |
| CNN-Transformer hybrid | Phase 2 |

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

> **Windows users:** If PyTorch installation fails with a path-too-long error,
> first run in an admin PowerShell:
> ```powershell
> New-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\FileSystem" -Name "LongPathsEnabled" -Value 1 -PropertyType DWORD -Force
> ```

### 2. Download the dataset

Download **PTB-XL v1.0.3** from PhysioNet (free account + data use agreement required):

```bash
wget -r -N -c -np --user=YOUR_USERNAME --ask-password \
  https://physionet.org/files/ptb-xl/1.0.3/ \
  -P ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3/
```

Verify the download:
```bash
python scripts/download_data.py \
  --data_path ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3/
```

Expected output: `ptbxl_database.csv` · `scp_statements.csv` · `records100/` 

### 3. Preprocess

Fits the per-lead normalizer on the training set and computes class weights:

```bash
python scripts/preprocess.py \
  --data_path ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3/ \
  --output_path data/processed/
```

Outputs:
- `config/norm_stats.npy` — per-lead mean/std
- `data/processed/pos_weight.npy` — class imbalance weights for BCEWithLogitsLoss

### 4. Train ResNet-1D

```bash
python scripts/train_resnet.py \
  --data_path ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3/
```

Best checkpoint saved to `checkpoints/resnet1d/best_model.pt` (based on val macro F1).

```

    ├── download_data.py     # validates dataset presence

```

> The dataset folders and checkpoints are gitignored — only source code is tracked.

## Classification target

5 diagnostic superclasses derived from SCP codes:

| Index | Class | Description |
|-------|-------|-------------|
| 0 | NORM | Normal ECG |
| 1 | MI | Myocardial Infarction |
| 2 | STTC | ST/T-wave Change |
| 3 | CD | Conduction Disturbance |
| 4 | HYP | Hypertrophy |

One record can be positive for multiple classes (multi-label).
