# ECG Classification — PTB-XL+

Multi-label ECG classification on the [PTB-XL](https://physionet.org/content/ptb-xl/1.0.3/) dataset (21,799 12-lead recordings, 500 Hz), comparing three deep learning architectures for 5-class diagnostic classification.

## Results (Test Set)

| Model | Params | AUC-ROC ↑ | F1 ↑ | AUPRC ↑ | Accuracy ↑ | Inference (ms/sample) |
|---|---|---|---|---|---|---|
| *ResNet-1D* | 8.9M | 0.9258 | *0.7409* | *0.8141* | *0.7643* | *0.214* |
| CNN-Transformer | 3.7M | *0.9264* | 0.7382 | 0.8081 | 0.7602 | 0.366 |
| PatchTST | 0.82M | 0.9100 | 0.7099 | 0.7728 | 0.7402 | 0.705 |

ResNet-1D and CNN-Transformer achieve near-identical macro AUC with no statistically significant difference (bootstrap, p > 0.05). CNN-Transformer is 2.4× smaller and trains 15% faster.

## Classification Target

5 diagnostic superclasses derived from SCP codes — multi-label (one record can be positive for multiple classes):

| Index | Class | Description |
|---|---|---|
| 0 | NORM | Normal ECG |
| 1 | MI | Myocardial Infarction |
| 2 | STTC | ST/T-wave Change |
| 3 | CD | Conduction Disturbance |
| 4 | HYP | Hypertrophy |

## Architectures

### ResNet-1D (models/resnet1d.py)
Residual convolutional network with Squeeze-and-Excitation attention and Stochastic Depth (DropPath).
- Stem: Conv(12→64, k=15, s=2) → BN → ReLU → MaxPool(k=4, s=4) → 625 timesteps
- 4 stages × 2 ResBlocks; channels [64, 128, 256, 512]
- Head: GlobalAvgPool → Dropout(0.3) → Linear(512, 5)
- ~8.9M parameters

### CNN-Transformer (models/cnn_transformer.py)
Depthwise-separable CNN front-end for local feature extraction + Transformer encoder for long-range context. Reduces sequence length 5000→313 before attention.
- ~3.7M parameters

### PatchTST (models/patchtst.py)
Channel-independent patch Transformer. Each lead split into 100 patches of 50 samples (100 ms at 500 Hz); patches processed by a 4-layer Transformer, lead representations aggregated by mean pooling.
- ~0.82M parameters

## Setup

### 1. Install dependencies

bash
pip install -r requirements.txt


> *Windows:* If PyTorch installation fails with a path-too-long error, run in an admin PowerShell:
> powershell
> New-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\FileSystem" -Name "LongPathsEnabled" -Value 1 -PropertyType DWORD -Force
> 

### 2. Download the dataset

Download *PTB-XL v1.0.3* from PhysioNet (free account + data use agreement required):

bash
wget -r -N -c -np --user=YOUR_USERNAME --ask-password \
  https://physionet.org/files/ptb-xl/1.0.3/ \
  -P ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3/


Required contents: ptbxl_database.csv · scp_statements.csv · records500/

### 3. Preprocess

Fits the per-lead normalizer on the training set (streaming Welford algorithm, ~O(1) RAM) and computes per-class BCE weights:

powershell
python scripts/preprocess.py `
    --data_path "ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3" `
    --output_path data/processed/ `
    --sampling_rate 500


Outputs:
- config/norm_stats.npy — per-lead mean/std (shape (12,))
- data/processed/pos_weight.npy — per-class weights for BCEWithLogitsLoss

### 4. Train and evaluate (notebooks)

Training, evaluation, and visualization are done through Jupyter notebooks in notebooks/:

| Notebook | Description |
|---|---|
| resnet1d.ipynb | ResNet-1D training, evaluation, and figures |
| cnn_transformer.ipynb | CNN-Transformer training, evaluation, and figures |
| patchtst.ipynb | PatchTST training, evaluation, and figures |
| comparacion_modelos.ipynb | Multi-model comparison and bootstrap statistical tests — requires the three model notebooks to have been run first |

Each model notebook produces: ROC curves, precision-recall curves, confusion matrices, training curves, and per-class metric bar charts, saved to results/figures/.
